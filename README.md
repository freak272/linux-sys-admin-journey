# linux-sys-admin-journey
This repository documents my journey to becoming a Linux System Administrator through hands-on practice, real-world labs, and clear explanations of how systems actually work
Goals
Build a strong foundation in Linux system administration
Understand how the operating system behaves internally
Develop the ability to troubleshoot and solve real problems
Transition into advanced areas like networking, security, and cloud
🧠 Learning Approach

My learning is based on three core principles:

1. Understand, Don’t Memorize

I break down commands and concepts to understand what they actually do.

2. Learn by Doing

I perform hands-on labs, experiments, and real system interactions.

3. Explain Clearly

If I can explain it simply, I understand it. Each topic is written in my own words.
